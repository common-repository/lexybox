Simple lightbox script to display images and videos on your website in a nice environment.

== Description ==

LexyBox is a simple, but powerful Lightbox script capable of displaying images or videos in a beautiful environment. But it is not just an ordinary lightbox, it can learn new things if you know how to script a bit. If you don't, it will surely make your visitors stay on your website more enjoyable. Learn more on the official [LexyBox website](https://lexybox.leximo.cz/).

== Supported media ==

- Images: (jpeg, jpg, gif, png, svg, webp)
- Video services: (youtube.com, vimeo.com)
- Local videos: (mp4, ogg, webm)
- [Add your own](https://lexybox.leximo.cz/#services)

== Features ==

- Gradual loading for large galleries
- Three built in transition effects
- Keyboard and touch control
- Able to learn new things
- Three built in themes
- Fully responsive
- Autoplay
- [Other things](https://lexybox.leximo.cz/)
